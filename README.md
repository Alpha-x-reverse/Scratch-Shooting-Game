# Scratch Shooting Game

## 📖 About

This is a 2D shooting game created in Scratch. The player controls a character that shoots enemies while avoiding losing all lives. The project was built to practice game development, programming logic, collision detection, and animations.

---

## 🎮 Features

- Player movement
- Shooting mechanics
- Skeleton enemy
- Explosion animation
- Lives system
- Score tracking
- Collision detection

---

## 🕹️ Controls

- Arrow Keys → Move
- Space Bar → Shoot

## 🚀 How to Run

1. Download the `.sb3` file.
2. Open Scratch.
3. Click **File → Load from your computer**.
4. Select the project file.
5. Click the green flag to start playing.
